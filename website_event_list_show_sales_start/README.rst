=================================================
Website Event: Show 'Sales Start' Info for Events
=================================================

* Show ticket sale start date in frontend event listing
* In case of multiple ticket types, the earlies Sales Start date is shown

Configuration
=============
* Add a ticket type to an event

Usage
=====
* Just navigate to /event to see the new field
